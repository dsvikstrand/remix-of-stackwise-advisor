# Google OAuth Reviewer Recording Quickcheck

Status: `active`  
Purpose: minimal checklist for recording the Google reviewer demo.

## Record These Steps
a1) [todo] Open `https://bleup.app/` and pause on the YouTube import explainer.

a2) [todo] Open `https://bleup.app/privacy` and pause on the Google/YouTube section.

a3) [todo] Sign in to Bleu.

a4) [todo] Open `https://bleup.app/subscriptions`.

a5) [todo] Pause on the `Connect YouTube` button and the surrounding explainer text.

a6) [todo] Click `Connect YouTube`.

a7) [todo] Pause on the Google consent screen so the reviewer can read:
- app name: `Bleup`
- requested permission

a8) [todo] Complete the connect flow and return to Bleu.

a9) [todo] Show the connected state and click `Import from YouTube` if that flow appears.

a10) [todo] Pause on the imported creator list / import result.

a11) [todo] Click `Disconnect` and show that the user can remove access.

## Recording Rules
b1) [todo] Keep the video short, ideally `1-2` minutes.

b2) [todo] Pause `2-4` seconds on the important screens.

b3) [todo] Screen-only is acceptable if the flow is clear.

b4) [todo] If you add narration or captions, keep them literal:
- read-only access
- import creators the user already follows
- personalize feed setup
- disconnect any time

## After Recording
c1) [todo] Save the video file or upload it somewhere stable.

c2) [todo] Record:
- date
- video location
- which Google account was used
